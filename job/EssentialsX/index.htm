Project EssentialsX

PSA: Do not use Mohist. It includes dangerous and potentially malicious behaviour. Click here for more information.

The essential plugin for Minecraft servers.

EssentialsX officially supports the CraftBukkit, Spigot and Paper (recommended) server software.

  • 1.18.1, 1.17.1 - EssentialsX actively develops against and supports these versions.
  • 1.8.8, 1.9.4, 1.10.2, 1.11.2, 1.12.2, 1.13.2, 1.14.4, 1.15.2, 1.16.5 - these versions are still supported, but are not a priority for us, and may be dropped in a future release.

We strongly recommend using the latest version of Paper. Paper includes countless bug fixes and performance improvements over Spigot and CraftBukkit, including fixing some non-vanilla behaviours in Spigot and CraftBukkit.
EssentialsX includes significant improvements when run on Paper which are impossible to achieve on Spigot or CraftBukkit, such as more efficient async teleports and support for modern gameplay features.

EssentialsX does not support the following Minecraft versions and server software:

  • 🛑 Do not use Mohist. Mohist tricks users into deleting official EssentialsX jars and installing unofficial modified software. Click here for more information.
  • 🛑 Akarin, Yatopia and some other "high performance" Paper forks. These servers include dangerous non-standard behaviour, which can not only impact EssentialsX's functionality but also cause damage to your worlds. If you *need* a high-performance fork, you should consider using Tuinity.
  • 🛑 Magma modifies EssentialsX bytecode at runtime and includes hacks to fix other bugs in Magma. If you encounter an issue with EssentialsX on Magma, please contact them for support.
  • 🛑 Other Forge/Fabric hybrid server software, including Cauldron, Thermos, CatServer etc. - the Bukkit API does not properly support mods, and using Bukkit plugins on modded Forge/Fabric servers will cause significant problems.
    For Forge servers, use SpongeForge with Nucleus for plugin support and a complete Essentials replacement, or consider a mod designed for Forge such as FTB Essentials.
  • 🛑 1.7.10 or below - you should use the original Essentials.

We will not be able to provide support if you use any of the software or versions listed above.
